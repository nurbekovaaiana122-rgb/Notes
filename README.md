# 📝 Notes App

Современное веб-приложение для создания и управления заметками.  
Простой интерфейс, быстрая работа и автоматическое сохранение данных.

---

## ✨ Возможности

- ➕ Добавление новых заметок
- 🗑️ Удаление заметок
- ✏️ Редактирование текста
- 💾 Автоматическое сохранение в localStorage
- 📱 Адаптивный дизайн (мобильная версия)
